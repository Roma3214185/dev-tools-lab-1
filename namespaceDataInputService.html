Detailed Description

Contains a set of functions to validate individual pieces of user input.

This namespace provides a modular, reusable validation system. All functions return detailed error messages describing exact reasons for failed validation (length mismatch, invalid characters, etc.).

ValidationResult DataInputService::validateUserInput ( const UserInput &  input,
const Config &  cfg = Config()

Validates all fields of a UserInput structure.

Validation runs in the following order:

  1. nameValidDetailed()
  2. emailValidDetailed()
  3. passwordValidDetailed()
  4. tagValidDetailed()

If a field fails validation, no further fields are checked.

Parameters
inputThe structure containing user-supplied name, email, password, and tag.
cfgValidation configuration.
Returns
ValidationResult The first validation error encountered, or success if all fields are valid.
